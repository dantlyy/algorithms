# Репозиторій лабораторних робіт з алгоритмів

Цей репозиторій містить лабораторні роботи по алгоритмах, розділені на категорії:

1. Quadratic_Sorting_Algorithms - Сортування вибором та вставками
2. Logarithmic_Sorting_Algorithms - Сортування злиттям та швидке сортування
3. Heap_Sorting - Сортування купою (Heap) як нестійкий ефективний метод
4. Hash_Tables - Відкрите та закрите хешування
5. Graph_MST - Мінімальне кістякове дерево: алгоритми Пріма та Крускала
6. Graph_Shortest_Path - Пошук найкоротшого шляху: алгоритми Дейкстри та Флойда
